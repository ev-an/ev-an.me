HotelBooking

A mock hotel booking app built using PHP and MySql for the back-end and SASS and JQuery for the front-end. The app allows users to register accounts to the database, while checking for problematic inputs and returning errors to the user where applicable. The app checks for double bookings by a single user and calculates price based on hotel, number of rooms and number of nights.

A quiz app build to test my knowledge of PHP form handling, functions and arrays.
